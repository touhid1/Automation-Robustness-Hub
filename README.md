## 🛠️ Framework Setup Instructions | ফ্রেমওয়ার্ক সেটআপ নির্দেশাবলী

### Playwright (Recommended) ⭐
```bash
# Initialize project
npm init playwright@latest

# Install dependencies
cd playwright-example
npm install

# Run tests
npx playwright test

# Debug mode
npx playwright test --debug
```

### Cypress 🌲
```bash
# Install Cypress
npm install cypress --save-dev

# Open Cypress
npx cypress open

# Run tests
npx cypress run
```

### Selenium 🌐
```bash
# Install Selenium
npm install selenium-webdriver

# Install WebDriver
npm install webdriver-manager

# Download WebDriver
npx webdriver-manager update
```
