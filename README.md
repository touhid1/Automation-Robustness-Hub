## How To Use This Export

1. Start with the CSV file if you want a database-style overview.
2. Open the markdown files in the `🤖 QA Automation Robustness Hub/` folder for detailed notes.
3. Work through challenges in priority order or by difficulty.
4. Use the framework comparison guide to choose between Playwright, Cypress, and Selenium.

## Suggested Learning Path

- Start with `Verify Account - Input Validation`
- Continue with medium challenges such as `Dynamic Table`, `Tags Input Box`, and `Multi-Level Dropdown Navigation`
- Finish with harder DOM-focused topics like `Sortable List`, `Nested iFrame`, and `Shadow DOM`
